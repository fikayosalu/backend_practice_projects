# TASK TRACKER

## DESCRIPTION

The task tracker is a command line interface that allows users to perform CRUD (Create, Read, Update, and Delete) operations on their tasks.

## HOW TO USE IT

### Run The Project

```bash
 npm run start
```

### Add A Task

```bash
 task-cli$ add {task-description}
```

### Update A Task

```bash
 task-cli$ update {task-id} {task-description}
```

### Delete A Task

```bash
 task-cli$ delete {task-id}
```

### Change Task status

```bash
 task-cli$ mark-in-progress {task-id}
 task-cli$ mark-done {task-id}
```

### List Available Tasks

```bash
 task-cli$ list
 task-cli$ list in-progress
 task-cli$ list done
```
